# openai-assistant-api-template
Chat with OpenAI Assistant using streamlit ChatUI 

# Installation
Python package
>pip install -r requirements.txt

Create file .env in your project directory and insert your OpenAPI key
>OPENAI_API_KEY=  
>ASSISTANT_ID=  
>THREAD_ID=  
>VECTOR_ID=  

# Run
start streamlit chatUI
>python src/main.py
